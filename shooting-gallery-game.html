Shooting Gallery Game

The shooting gallery game for iOS, except it uses fish under the sea as targets. You have a set amount of time to try and touch as many fish as you can. Living fish will add to your score while dead ones will subtract from it. The game also comes with a few nice sound effects.

Stack: Swift · SpriteKit Repo
